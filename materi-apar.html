Materi Penggunaan APAR

Alat Pemadam Api Ringan di Laboratorium

Alat Pemadam Api Ringan atau APAR merupakan peralatan darurat yang digunakan untuk memadamkan kebakaran pada tahap awal. Setiap pengguna laboratorium perlu memahami cara penggunaannya dengan benar dan sesuai prosedur keselamatan.

Sebelum digunakan, lakukan pemeriksaan kondisi APAR. Pastikan pin pengaman masih terpasang dan segel tidak rusak. Periksa manometer, dan pastikan jarum indikator berada pada zona hijau. Jika berada di luar zona tersebut, segera laporkan kepada petugas untuk dilakukan pemeriksaan sebelum digunakan

Pegang APAR dengan posisi stabil. Satu tangan pada tuas atau gagang, dan tangan lainnya menopang bagian bawah atau memegang selang untuk menjaga kendali saat penyemprotan.

Sebelum menggunakan APAR, tarik pin pengaman untuk membuka kunci tuas. Arahkan nozzle atau selang ke bagian dasar api, bukan ke bagian atas nyala api. Tekan tuas secara perlahan untuk mengeluarkan media pemadam. Lakukan uji semprot singkat terlebih dahulu untuk memastikan APAR berfungsi dengan baik. Lakukan penyemprotan dengan gerakan menyapu dari sisi ke sisi hingga api benar-benar padam.

Setelah digunakan, pasang kembali selang pada tempatnya dan kembalikan pin pengaman. Laporkan penggunaan APAR kepada petugas untuk dilakukan pemeriksaan dan pengisian ulang sebelum dipasang kembali.

Gunakan APAR dengan benar. Tindakan yang cepat dan terkontrol dapat mencegah kebakaran meluas.
